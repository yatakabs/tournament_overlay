  1. Head Distance Travelled のインストール (必須)
  2. HttpHDTStatus のインストール (必須)
  3. HDT 対応 tournament-overlay の設定 (OBS Studio) (必須)
  4. HDTCounter のインストール (任意)

コピー後の HttpHDTStatus.dll が下記の場所に存在する状態となる。
C:\Program Files (x86)\Steam\steamapps\common\Beat Saber\Plugins\HttpHDTStatus.dll 不要なファイルを削除する Mod の干渉を防ぐため、もし Beat Saber の Plugins フォルダに下記のファイルがあれば、それを削除する。
  • BeatSaberHttpStatus.dll
  • HttpStatus.dll
 HDT 対応 tournament-overlay の設定 (OBS Studio) OBS に Browser Source を追加する
  1. URL には http://bstoverlay.yatakabs.com/ を指定する。 Beat Saber を実行するマシンと OBS を実行するマシンが同一であれば、SSL を有効にした https://bstoverlay.yatakabs.com/ も利用可能。
  2. Width には、配信の画面解像度の横幅を指定する。
    • 配信解像度が 1920x1080 (1080p) であれば、1920
    • 配信解像度が 1280x720 (720p) であれば、1280
  3. Height には、配信の画面解像度の縦幅を指定する。
    • 配信解像度が 1920x1080 (1080p) であれば、1080
    • 配信解像度が 1280x720 (720p) であれば、720
OBS に Browser Source を前面に移動し、画面全体に表示する。
  1. 追加した Browser Source は、Beat Saber の Game Capture Source より前面に表示されるよう順番を調整する。
  2. 追加した Browser Source は、画面全体に等倍で表示されていることを確認する。 HDT 対応 tournament-overlay は、原則としてこの指示通り、画面上部に表示されるよう配置してください。
HDTCounter のインストール

HeadDistanceTravelled は標準で頭上に移動距離が表示されるため自身でリアルタイムに移動距離の確認ができません。 この HDTCounter Mod を利用すると、Counter+ のカウンターとして移動距離が表示できるようになります。

HttpHDTStatus.dll のインストール HttpHDTStatus のダウンロードと配置 Beat Saber のバージョンに応じてダウンロード 1.2*.* の場合 Github から最新版の zip ファイルをダウンロード。
(HDTCounter-1.0.0-bs1.20.0-8261247.zip) それ以前のバージョン 非対応です。 ダウンロードした zip ファイルを解凍 解凍したファイルを Plugins フォルダへ移動

Beat Saber のインストールパスが下記の場合
C:\Program Files (x86)\Steam\steamapps\common\Beat Saber
コピー後の HDTCounter.dll が下記の場所に存在する状態となる。
